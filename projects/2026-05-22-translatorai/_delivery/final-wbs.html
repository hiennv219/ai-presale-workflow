- Khởi tạo database SQLite cục bộ thành công.
M3: APIs Dịch thuật & Cài đặt Tích hợp Google Translate API, OpenAI GPT-4o API và hoàn thiện Preferences Panel cùng SQLite lưu trữ lịch sử. - Dịch thành công qua Google Translate and OpenAI GPT-4o.
- Lưu cấu hình OpenAI API Key cá nhân vào Preferences panel.
- Lịch sử dịch thuật được lưu vào SQLite cục bộ và hiển thị/xóa được từ giao diện cài đặt.
M4: Kiểm thử, Tối ưu & Đóng gói Kiểm thử hiệu năng, tối ưu bộ nhớ, kiểm tra bảo mật, notarize và đóng gói thành file .dmg phân phối. - Không có lỗi nghiêm trọng (Critical/Blocker).
- CPU tiêu hao khi dịch thấp (<5% peak), bộ nhớ RAM tĩnh chạy ngầm <30MB.
- Notarize thành công từ Apple và xuất file cài đặt TranslatorAI.dmg chạy tốt trên macOS 14+.
